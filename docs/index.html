핵심 기술적 설계 및 방어 전략 Core Architectural Advancements

기존 이진 트리 패리티 머신(TPM)의 수학적 취약점을 보완하고 크로스 플랫폼 호환성을 확보하기 위한 세 가지 중점 혁신입니다. Three strategic design iterations that eliminate mathematical vulnerabilities of traditional TPMs and guarantee cross-platform compatibility.

하이브리드 활성화 모드 (Hybrid Activation) Hybrid Activation Mode

순수 카오스 비선형 활성화 함수(Sine 등)는 학습 방향성을 깨뜨려 수렴을 방해합니다. DeepEnigma는 동기화 시점엔 이진 부호 함수(Sign)를 사용하여 빠른 수렴을 유도하고, 키 도출 시점에 비선형 카오스 맵을 거쳐 키를 경화(Hardening)하는 하이브리드 설계를 도입했습니다.

Pure chaotic activation functions break monotonic gradients and disrupt convergence. DeepEnigma resolves this by using stable sign activation for synchronization, then applying non-linear chaotic mappings for key-hardening only during the final derivation phase.

정수 기반 카오스 텐트 맵 (Integer Tent Map) Integer-Only Chaotic Tent Map

ARM(모바일/임베디드)과 x86(서버/PC) 등 이기종 하드웨어 간 부동 소수점(float) 연산 정밀도 미세 차이는 카오스 계에서 나비효과를 일으켜 키 불일치를 만듭니다. 우리는 100% 정수 연산만으로 작동하는 결정성 텐트 맵(Integer Tent Map)을 개발해 플랫폼 무관 비트 단위 일치를 보장합니다.

Floating-point precision differences between architectures (e.g. ARM vs x86) cause chaotic values to diverge. We utilize an integer-only deterministic chaotic Tent Map, guaranteeing bit-for-bit key identity across disparate physical devices.

ZKP 상호 인증 (ZKP Mutual Authentication) Zero-Knowledge Mutual Authentication

기존 신경망 키 교환은 도청자 공격(Eve)엔 강하지만 중간자 공격(MitM)엔 무방비했습니다. 동기화 전 단계에 Fiat-Shamir 휴리스틱 기반의 해시 해밀턴식 영지식 증명(ZKP) 프로토콜을 탑재하여 비밀 공유 값을 노출하지 않고 상대방의 신원을 검증함으로써 MitM을 사전 차단합니다.

Traditional neural cryptography is vulnerable to Man-in-the-Middle (MitM) attacks. We integrate a lightweight hash-based Zero-Knowledge challenge-response protocol prior to weight synchronization, validating identity without leaking secrets.
